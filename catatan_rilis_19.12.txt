Di rilis ini, versi 19.12, Sediakan API untuk menyediakan data informasi publik dalam format json.
Selain itu, rilis ini juga berisi perbaikan lain yang diminta Komunitas SID.

Terima kasih pada Zeppy Shiddiq, Nur Hidayat dan Il Ham yang terus berkontribusi.

Lengkapnya, isi rilis versi 19.12 adalah sebagai berikut:

1. #2444: Perbaiki database supaya sesuai sql_mode STRICT_TRANS_TABLES.
2. #2455: Sekarang pengaturan data keuangan dapat ditampilkan untuk widget keuangan. [bug-fix]
3. Perbaiki sekarang bisa kembali tambah, ubah dan hapus rumah tangga. [bug-fix]
4. #2458: Sekarang scroll horizontal tampil kembali untuk tabel yang lebar, seperti daftar penduduk. [bug-fix]
5. #2464: Sediakan data asuransi sebagai pilihan statistik kependudukan untuk menu statis.
6. #2468: Pindahkan submenu Informasi Publik ke menu Sekretariat.
7. #2469: Pindahkan kolom kategori informasi publik dari bagian attr ke kolom utama di tabel dokumen. [teknis]
8. #2463: Batasi dokumen aktif saja yg tampil di daftar informasi publik di web.
9. #2470: Tampilkan SK Kades dan Perdes sebagai dokumen Informasi Publik dengan kategori 'Informasi Setiap Saat'.
10. #2442: Ekspor keseluruhan data informasi publik.
11. #2473: Perbaiki tampilan form Ubah Data KK pada layar kecil.
12. #2414: Perbesar nilai klasifikasi supaya bisa lebih dari nilai 999,99.
13. #2425: Sekarang gambar artikel yg disipkan melalui editor tampil pada url tulisan dan id. Gambar juga sekarang tampil dengan proporsi yg benar. [bug-fix]
14. Sekarang data peta dalam format GPX/KML dapat diimpor ke peta kantor/wilayah desa/dusun/rw/rt.
15. #2454: Ekspor perubahan data informasi publik sejak tanggal tertentu.
16. #1550: Sekarang peta Lokasi/Garis/Area bisa diimpor dari file GPX/KML dan bisa diekspor ke file GPX/KML.
17. Fitur Pemetaan > Peta dan Pemetaan > Pengaturan Peta telah banyak dilengkapi, termasuk penampilan statistik penduduk per wilayah administratif di peta.
18. #2502: Pengaturan > Database > Kosongkan DB sekarang tidak menghapus tabel tweb_penduduk_asuransi atau klasifikasi_surat. [bug-fix]
19. #2202: Citra satelit dari Mapbox sekarang bisa ditampilkan di peta.
20. Sediakan API untuk menyediakan data informasi publik dalam format json.
21. Unduh berkas dokumen melalui controller dan tidak lagi secara langsung dari lokasi di server. [security-fix]


Selalu diingatkan apabila meng-upgrade (yaitu bukan install baru) dari versi SEBELUM versi 19.12, struktur database dan/atau folder desa perlu diubah dengan menjalankan fitur Database > Migrasi DB. Fitur migrasi ini melakukan perubahan database dan/atau folder desa yang diperlukan.
